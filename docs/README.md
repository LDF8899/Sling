## 项目简介

Sling 是一个基于微服务架构的蛇类管理与应急服务平台。旨在解决蛇类识别困难、紧急救援响应慢、医疗资源匹配不准以及风险预警缺失等核心问题，为公众和救援人员提供技术支持。

### 核心功能
- 蛇类AI识别
- 应急处理指导
- 风险预警
- 医院/血清查询
- 知识图谱
