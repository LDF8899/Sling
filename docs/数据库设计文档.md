## 二、核心表结构

### 2.1 snake_info — 蛇类物种信息（主表）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| snake_id | bigint PK | 是 | 自增主键 |
| snake_name | varchar(100) UNI | 是 | 中文名 |
| family | varchar(50) | 否 | 科名（眼镜蛇科、蝰科等） |
| genus | varchar(50) | 否 | 属名（环蛇属、竹叶青蛇属等） |
| latin_name | varchar(200) | 否 | 拉丁学名（含命名人） |
| characteristics | text | 是 | 形态特征描述 |
| toxicity_level | int | 是 | 毒性等级：0=无毒 1=轻度 2=中度 3=重度 |
| toxin_type | varchar(50) | 否 | 毒素类型：神经毒素/血液毒素/细胞毒素/混合毒素/无毒 |
| danger_level | varchar(20) | 否 | 危险梯队：无毒/轻度/中度/重度 |
| habitat_info | text | 否 | 栖息环境 |
| distribution | text | 否 | 分布省份，逗号分隔 |
| conservation_status | varchar(50) | 否 | 保护等级：无危/易危/濒危/数据缺乏 |

**当前数据：** 189 种，覆盖 18 科 ~55 属
**毒性分布：** 重度 94 / 中度 3 / 轻度 2 / 无毒 90

### 2.2 snake_emergency_info — 蛇类急救信息

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | bigint PK | 是 | 自增主键 |
| snake_name | varchar(100) UNI | 是 | 蛇类名称 |
| snake_alias | varchar(200) | 否 | 别名/俗称 |
| venom_type | varchar(50) | 否 | 毒液类型 |
| symptom_description | text | 否 | 中毒症状描述 |
| emergency_treatment | text | 否 | 紧急处理方法 |
| medical_attention | text | 否 | 就医注意事项 |
| latin_name | varchar(200) | 否 | 拉丁学名 |
| forbidden_actions | text | 否 | 严格禁止的行为 |
| serum_type | varchar(200) | 否 | 解毒血清类型 |
| hospital_department | varchar(200) | 否 | 建议就医科室 |
| image_url | varchar(300) | 否 | 蛇类图片 URL |
| del_flag | tinyint | 否 | 逻辑删除：0=正常 1=删除 |

**当前数据：** 18 条（字段大量 NULL，待补全）

### 2.3 hospital_info — 医院信息

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| hospital_id | bigint PK | 是 | 自增主键 |
| hospital_name | varchar(100) | 是 | 医院名称 |
| address | varchar(255) | 是 | 详细地址 |
| latitude | decimal(10,6) | 否 | 纬度（Geo 索引用） |
| longitude | decimal(10,6) | 否 | 经度（Geo 索引用） |
| contact_info | text | 否 | 联系方式 |
| hospital_type | varchar(50) | 否 | 医院类型 |
| serum_availability | tinyint(1) | 否 | 是否有血清库存 |
| emergency_department | tinyint(1) | 否 | 是否有急诊科 |
| snake_venom_treatment | tinyint(1) | 否 | 是否能治疗蛇毒 |
| snake_venom_keywords | varchar(255) | 否 | 擅长治疗的蛇毒类型 |

### 2.4 serum_inventory — 血清库存

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| inventory_id | bigint PK | 是 | 自增主键 |
| hospital_id | bigint FK | 是 | 关联 hospital_info |
| snake_id | bigint FK | 是 | 关联 snake_info |
| serum_amount | int | 是 | 库存数量 |
| serum_expiry_date | date | 否 | 过期日期 |
| inventory_metadata | text | 否 | 额外元数据 JSON |

### 2.5 snake_active_area — 蛇类活跃区域

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| area_id | bigint PK | 是 | 自增主键 |
| area_name | varchar(100) | 是 | 区域名称 |
| latitude | decimal(10,6) | 否 | 中心纬度 |
| longitude | decimal(10,6) | 否 | 中心经度 |
| radius | int | 否 | 影响半径（米） |
| snake_id | bigint FK | 是 | 关联 snake_info |
| active_count | int | 否 | 活跃度 |

### 2.6 recognition_record — 识别记录

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| record_id | bigint PK | 是 | 自增主键 |
| user_id | bigint FK | 是 | 关联 user_info |
| snake_id | bigint FK | 否 | 识别出的蛇类 ID |
| image_path | varchar(255) | 是 | 上传图片路径 |
| recognition_time | timestamp | 否 | 识别时间 |
| recognition_result | varchar(255) | 是 | 识别结果 |
| recognition_metadata | text | 否 | 元数据 JSON（模型版本、置信度等） |

### 2.7 预警相关表

**warning_area** — 预警区域：area_id, area_name, description, boundary_coordinates(JSON), snake_species(JSON), warning_level(1-4)

**warning_record** — 预警记录：id, area_id(FK), warning_content, warning_time, is_valid

**warning_rule** — 预警规则：rule_id, snake_id(FK), active_threshold, warning_message, rule_metadata(JSON)

### 2.8 用户权限表

**user_info** — 用户：user_id, username, password, email, wechat_openid, wechat_unionid, wechat_nickname, wechat_avatar, phone, avatar_url, extra_info(JSON)

**role** — 角色：role_id, role_name(ADMIN/MODERATOR/USER/VIP), role_description

**user_role** — 用户角色关联：id, user_id(FK), role_id(FK)

**user_permission** — 权限：permission_id, user_id(FK), permission_name, permission_description

### 2.9 其他表

**emergency_qa_cache** — AI问答缓存：id, question, answer, created_time, updated_time

**user_action_log** — 行为日志：log_id, user_id(FK), action_type, action_time, action_detail, action_metadata(JSON)

## 三、表关系

```
snake_info (主表)
  ├── snake_emergency_info (1:1, 通过 snake_name 关联)
  ├── snake_serum_match (1:N, 通过 snake_id)
  ├── snake_active_area (1:N, 通过 snake_id)
  ├── serum_inventory (1:N, 通过 snake_id)
  ├── recognition_record (1:N, 通过 snake_id)
  └── warning_rule (1:N, 通过 snake_id)

hospital_info
  └── serum_inventory (1:N, 通过 hospital_id)

user_info
  ├── user_role (1:N) ── role (N:1)
  ├── user_permission (1:N)
  ├── recognition_record (1:N)
  └── user_action_log (1:N)

warning_area
  └── warning_record (1:N)
```
