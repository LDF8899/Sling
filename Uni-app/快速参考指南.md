## 🔍 常见问题排查

### 问题 1: API 请求失败
**检查清单**:
- [ ] 后端服务是否启动
- [ ] BASE_URL 是否正确
- [ ] 防火墙是否开放端口
- [ ] 跨域配置（开发环境）

**解决方案**:
```javascript
// 临时解决：在微信开发者工具中
详情 -> 本地设置 -> 勾选「不校验合法域名」
```

### 问题 2: 样式不生效
**可能原因**:
- `<style>` 标签缺少 `scoped` 属性
- 使用了不支持的 CSS 属性
- 选择器优先级不够

**解决方案**:
```vue
<style lang="scss" scoped>
/* 添加 !important 提高优先级 */
.custom-class {
  color: red !important;
}
</style>
```

### 问题 3: Pinia 状态不更新
**检查点**:
- 是否正确使用 `useUserStore()`
- 是否调用了 action 方法
- 数据是否保存到本地存储

**调试方法**:
```javascript
console.log('当前用户状态:', userStore.$state)
```

### 问题 4: 页面跳转报错
**常见错误**:
- 页面路径未注册（检查 pages.json）
- TabBar 页面使用了 navigateTo

**正确用法**:
```javascript
// 普通页面
uni.navigateTo({ url: '/pages/recognition/recognition' })

// TabBar 页面
uni.switchTab({ url: '/pages/index/index' })
```

## 📱 页面开发模板

创建新页面时，复制此模板：

```vue
<template>
  <view class="page-container">
    <!-- 顶部导航 -->
    <view class="navbar">
      <view class="nav-back" @click="goBack">← 返回</view>
      <view class="nav-title">页面标题</view>
      <view class="nav-placeholder"></view>
    </view>
    
    <!-- 主内容区 -->
    <scroll-view class="main-content" scroll-y enhanced>
      <!-- 内容写在这里 -->
    </scroll-view>
  </view>
</template>

<script setup>
import { ref, onMounted } from 'vue'
import api from '@/utils/api'

// 响应式数据
const loading = ref(false)
const dataList = ref([])

// 方法
const loadData = async () => {
  try {
    loading.value = true
    const response = await api.xxx.getData()
    if (response.code === 200) {
      dataList.value = response.data
    }
  } catch (error) {
    console.error('加载失败:', error)
  } finally {
    loading.value = false
  }
}

const goBack = () => {
  uni.navigateBack()
}

onMounted(() => {
  loadData()
})
</script>

<style lang="scss" scoped>
.page-container {
  min-height: 100vh;
  background: #f5f5f5;
}

.navbar {
  position: sticky;
  top: 0;
  z-index: 100;
  background: white;
  padding: 10px 15px;
  display: flex;
  justify-content: space-between;
  align-items: center;
  box-shadow: 0 2px 8px rgba(0,0,0,0.05);
}

.main-content {
  height: calc(100vh - 50px);
  padding: 15px;
}
</style>
```
