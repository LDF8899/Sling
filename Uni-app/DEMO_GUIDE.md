## 📱 演示环境准备

### 1. 后端服务检查

确保以下服务已启动：

```bash
# Windows - 使用启动脚本
.\启动脚本.bat

# 或手动启动各个微服务
- snake-gateway        (端口：8888)

- recognition-service  (端口：8002)
- emergency-service    (端口：8003)
- hospital-service     (端口：8004)
- warning-service      (端口：8005)
```

### 2. 前端运行

```bash
cd Uni-app
npm run dev:mp-weixin
```

在微信开发者工具中打开并编译。

## ⚠️ 常见问题应对

### Q1: 图片上传失败？
**应对**: 
- 检查后端服务是否启动
- 确认 BASE_URL 配置正确
- 使用备用图片测试
- 展示录屏或截图

### Q2: 定位不准确？
**应对**:
- 解释模拟器限制
- 手动输入经纬度
- 展示真机演示视频
- 强调实际环境可用

### Q3: AI 识别慢？
**应对**:
- 解释网络传输时间
- 展示进度动画
- 说明后台处理流程
- 提供优化方案

### Q4: TabBar 图标缺失？
**应对**:
- 说明是临时占位符
- 展示图标生成脚本
- 提供正式图标方案
- 不影响功能演示
