| 等级 | 含义 |

| **NE** | 未予评估 (Not Evaluated) - 尚未评估保护状态 |
| **NT** | 近危 (Near Threatened) - 可能面临威胁 |
| **VU** | 易危 (Vulnerable) - 面临较高灭绝风险 |
| **EN** | 濒危 (Endangered) - 面临高度灭绝风险 |
| **CR** | 极危 (Critically Endangered) - 极度濒临灭绝 |
