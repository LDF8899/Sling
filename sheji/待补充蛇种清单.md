## 一、需要查的数据字段

每种蛇需要以下信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| snake_name | 中文名 | 银环蛇 |
| family | 科名 | 眼镜蛇科 |
| genus | 属名 | 环蛇属 |
| latin_name | 拉丁学名（含命名人） | Bungarus multicinctus (Boulenger, 1860) |
| characteristics | 形态特征（50-100字） | 体背黑白相间环纹，白环窄黑环宽... |
| toxicity_level | 毒性等级：0=无毒 1=轻度 2=中度 3=重度 | 3 |
| toxin_type | 毒素类型 | 神经毒素 |
| danger_level | 危险梯队：无毒/轻度/中度/重度 | 重度 |
| habitat_info | 栖息环境 | 平原丘陵溪沟边、稻田 |
| distribution | 分布省份（中国蛇用省份，非中国蛇用国家） | 广东,广西,福建,浙江 |
| conservation_status | 保护等级 | 无危/易危/濒危/数据缺乏 |

## 四、推荐数据来源

1. **The Reptile Database** (reptile-database.org) — 全球最权威的爬行动物分类数据库，可按属名检索完整物种列表
2. **中国科学院成都生物研究所 — 中国蛇类名录 (2024)** — 19科71属355种的最新官方名录
3. **《中国蛇伤救治专家共识 (2025)》** — 毒性分级和急救指南的权威来源
4. **中国两栖爬行动物多样性信息平台** — 分布数据和形态描述
