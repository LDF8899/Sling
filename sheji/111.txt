蛇灵 (SLING) 系统详细模块说明
项目概述
"蛇灵"(SLING) 是一款专为户外工作人员、户外运动爱好者打造的综合性蛇类识别与应急服务生态平台。系统以 “安全户外 + 实用服务 + 精准消费” 为核心，通过现代化微服务架构，构建高可用性、可扩展性强的户外生态体系。不仅为用户提供蛇类快速识别、预警信息获取、应急指导及医疗救助等核心安全服务，还新增户外装备商城模块，整合专业户外防护装备资源，满足用户安全出行与装备采购的双重需求，实现安全保障与商业价值的协同发展。
整体架构
系统采用 Spring Cloud Alibaba 微服务架构，主要包含以下组件：
前端：Vue 3 + Vite
后端：Spring Boot + Spring Cloud Gateway + Nacos
数据库：MySQL
AI 服务：火山引擎豆包模型（文本和视觉识别）
电商支撑：支付接口集成、订单管理系统

系统核心特性
AI 智能识别：利用先进的 AI 视觉模型对蛇类进行准确识别
实时预警系统：提供蛇类活跃区域的实时预警信息
应急响应机制：集成应急处理指南和紧急求助功能
医疗资源导航：快速定位蛇伤治疗医院并提供导航服务
多角色权限管理：支持普通用户和管理员的不同权限级别
跨平台兼容性：前端采用 Vue3 技术，支持多种设备访问
微服务架构：高可用、易扩展的微服务架构设计
场景化装备商城：聚焦户外安全场景，提供精准防护装备采购渠道，实现安全服务与消费需求的无缝衔接
一体化生态服务：整合 “识别 - 预警 - 应急 - 医疗 - 装备” 全链路服务，打造户外安全闭环
数据流转过程
用户数据流：用户注册 / 登录 → 认证授权 → 个性化服务（安全服务 + 商城服务）
识别数据流：图片上传 → AI 识别 → 结果返回 → 记录存储 → 防护装备推荐（可选）
预警数据流：位置信息 → 风险评估 → 预警生成 → 推送通知 → 高风险区域适配装备推荐（可选）
应急数据流：症状描述 → AI 分析 → 应急指南 → 医院推荐
医院数据流：位置查询 → 医院匹配 → 路径规划 → 导航服务
商城数据流：装备浏览 → 加入购物车 → 订单创建 → 支付完成 → 物流跟踪 → 评价反馈
模块详细说明（新增商城模块）
7. 户外装备商城模块 (mall-service)
文件路径：C:\Users\Asuka\Desktop\1\Sling\mall-service\
功能：
商品管理：蛇类防护装备、户外安全装备的信息维护（新增、编辑、上下架）
分类管理：按装备类型（防护类、急救类、通讯类等）、适用场景（徒步、露营、勘探等）进行分类
购物流程：商品浏览、详情查看、购物车操作、订单提交、在线支付
订单管理：订单状态跟踪（待支付、已支付、已发货、已收货、已完成）、订单取消、退款处理
物流对接：集成物流查询接口，实时展示物流信息
评价互动：用户对购买装备进行评分、评论，支持图片上传
营销功能：限时折扣、安全装备套餐组合、新用户专享优惠、积分兑换
数据统计：商品销量统计、用户消费行为分析、营销活动效果分析
核心控制器：MallController.java
详细 API 参数说明：
获取商品分类列表：
路径：GET /mall/category/list
参数：无
响应：分类 ID、分类名称、分类描述、分类图标 URL
获取商品列表：
路径：GET /mall/product/list
参数（查询参数）：
categoryId (可选)：分类 ID 筛选
keyword (可选)：商品名称关键词搜索
page (可选，默认 0)：页码
size (可选，默认 10)：每页大小
sort (可选)：排序方式（销量 desc/price asc/price desc）
获取商品详情：
路径：GET /mall/product/detail/{productId}
参数：路径参数
{productId}: 商品 ID（数字）
响应：商品基本信息、规格参数、价格、库存、图片列表、用户评价摘要
加入购物车：
路径：POST /mall/cart/add
参数：JSON 格式
json
{
  "userId": "用户ID",
  "productId": "商品ID",
  "quantity": "购买数量",
  "specification": "商品规格（可选）"
}
查看购物车：
路径：GET /mall/cart/list/{userId}
参数：路径参数
{userId}: 用户 ID（数字）
创建订单：
路径：POST /mall/order/create
参数：JSON 格式
json
{
  "userId": "用户ID",
  "cartItemIds": ["购物车项ID列表"],
  "receiptName": "收货人姓名",
  "receiptPhone": "收货人电话",
  "receiptAddress": "收货地址",
  "paymentType": "支付方式（微信/支付宝）"
}
订单支付：
路径：POST /mall/order/pay
参数：JSON 格式
json
{
  "orderId": "订单ID",
  "paymentType": "支付方式",
  "payAmount": "支付金额"
}
响应：支付链接 / 支付二维码信息
查询订单列表：
路径：GET /mall/order/list/{userId}
参数：
路径参数：{userId}: 用户 ID
查询参数：
status (可选)：订单状态筛选
page (可选，默认 0)：页码
size (可选，默认 10)：每页大小
查询物流信息：
路径：GET /mall/order/logistics/{orderId}
参数：路径参数
{orderId}: 订单 ID（数字）
提交商品评价：
路径：POST /mall/product/comment
参数：JSON 格式
json
{
  "userId": "用户ID",
  "productId": "商品ID",
  "orderId": "订单ID",
  "score": "评分（1-5分）",
  "content": "评价内容",
  "imageUrls": ["评价图片URL列表（可选）"]
}

业务流程（新增商城相关流程）
4. 装备采购流程
用户通过首页入口或识别结果页推荐进入装备商城
浏览分类商品或搜索目标装备，查看商品详情（参数、评价、价格）
选择规格与数量，加入购物车或直接购买
提交订单，填写收货信息，选择支付方式
完成在线支付，系统生成订单并同步至商家
商家发货后，用户可实时查询物流状态
收到装备后，用户可进行确认收货与评价反馈
5. 安全与消费联动流程
用户完成蛇类识别，系统返回识别结果（如剧毒蛇类）
系统自动推荐适配的防护装备（如防蛇靴、专用驱虫剂）
用户点击推荐链接直接跳转至对应商品详情页
支持快速加入购物车，简化采购流程
高风险预警区域用户访问系统时，首页弹窗展示热门防护装备套餐
