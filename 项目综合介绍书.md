## 一、项目概述

### 1.1 项目背景

**蛇灵 (SLING)** 是一个专为户外工作者和爱好者设计的智能化蛇类识别与应急处理系统。项目的诞生源于以下现实需求：

- **识别困难**：大多数人在野外无法准确识别蛇类是否有毒
- **预警缺失**：对蛇类活跃区域缺乏及时了解，增加遭遇风险
- **应急不当**：被蛇咬伤后缺乏正确的急救知识，可能危及生命
- **医疗资源信息不对称**：不知道哪些医院有对应血清，延误治疗时机

### 1.2 项目定位

SLING 项目通过现代化的微服务架构，构建一个**高可用、可扩展、智能化**的户外应急服务平台，实现：

```
识别 → 预警 → 应急 → 医疗 → 装备
```

全链路服务闭环，打造户外安全生态系统。

### 1.3 核心价值

| 价值维度 | 具体体现 |
|---------|---------|
| **安全性** | AI 智能识别 + 实时预警 + 科学应急指导 |
| **及时性** | 秒级识别响应 + 精准医院导航 |
| **专业性** | 权威蛇类数据库 + 医学应急方案 |
| **便捷性** | 小程序即开即用 + 一键求助 |
| **可靠性** | 微服务高可用架构 + 多冗余设计 |

### 1.4 目标用户

- **户外工作者**：林业工人、电力巡检员、地质勘探人员
- **户外运动爱好者**：徒步、登山、露营爱好者
- **农村居民**：生活在蛇类活跃区域的居民
- **应急救援人员**：需要蛇类知识的救援志愿者

### 2.2 微服务拆分原则

遵循**单一职责、高内聚低耦合**的设计思想：

| 服务名称 | 端口 | 职责描述 | 核心功能 |
|---------|------|---------|---------|
| **snake-gateway** | 8888 | API 网关 | 路由转发、权限校验、限流熔断 |
| **user-service** | 7006 | 用户服务 | 注册登录、权限管理、用户行为 |
| **recognition-service** | 7003 | 识别服务 | 图片上传、AI 识别、结果返回 |
| **emergency-service** | 9093 | 应急服务 | 伤口分析、AI 问答、应急指南 |
| **warning-service** | 8091 | 预警服务 | 风险区域分析、预警推送 |
| **hospital-service** | 8089 | 医院服务 | 医院查询、路径规划、血清匹配 |
| **snake-info-service** | 7004 | 信息服务 | 蛇类数据库、特征库管理 |
| **admin-service** | 8092 | 管理服务 | 后台管理、内容审核、系统配置 |

// VolcanoAIService.java
public String callVisionModel(String prompt, String imageUrl) {
    // 构建请求头
    HttpHeaders headers = new HttpHeaders();
    headers.setContentType(MediaType.APPLICATION_JSON);
    headers.set("Authorization", "Bearer " + apiKey);
    
    // 构建请求体
    Map<String, Object> requestBody = new HashMap<>();
    requestBody.put("model", "doubao-seed-1-6-vision-250815");
    requestBody.put("max_completion_tokens", 2000);
    
    List<Map<String, Object>> messages = new ArrayList<>();
    Map<String, Object> userMessage = new HashMap<>();
    userMessage.put("role", "user");
    
    // 文本 + 图片多模态输入
    List<Map<String, Object>> content = new ArrayList<>();
    content.add(Map.of("type", "text", "text", prompt));
    content.add(Map.of("type", "image_url", 
                       "image_url", Map.of("url", imageUrl)));
    
    userMessage.put("content", content);
    messages.add(userMessage);
    requestBody.put("messages", messages);
    
    // 发送请求
    ResponseEntity<Map<String, Object>> response = 
        restTemplate.postForEntity(API_URL, 
                                   new HttpEntity<>(requestBody, headers),
                                   Map.class);
    
    return parseResponse(response.getBody());
}

const compressImage = async (imagePath) => {
  const stat = await uni.stat(imagePath);
  const originalSize = stat.size;
  
  // 超过 500KB 才压缩
  if (originalSize > 500 * 1024) {
    const compressedPath = await uni.compressImage({
      src: imagePath,
      quality: 80  // 80% 质量
    });
    return { path: compressedPath.path, compressed: true };
  }
  
  return { path: imagePath, compressed: false };
};

// HospitalMapController.java
@PostMapping("/snake-venom/nearby")
public Result<List<HospitalVO>> searchNearbyHospitals(
    @RequestBody HospitalSearchRequest request) {
    
    // 1. 地址解析（如果是文字地址）
    if (request.isAddress()) {
        Coordinate coord = geocodeService.geocode(request.getAddress());
        request.setLatitude(coord.getLat());
        request.setLongitude(coord.getLng());
    }

    List<HospitalInfo> hospitals = hospitalMapper.selectByRadius(
        request.getLatitude(),
        request.getLongitude(),
        request.getRadius()
    );
    
    // 3. 计算距离并排序
    List<HospitalVO> result = hospitals.stream()
        .map(h -> {
            HospitalVO vo = convertToVO(h);
            double distance = calculateDistance(
                request.getLatitude(), request.getLongitude(),
                h.getLatitude(), h.getLongitude()
            );
            vo.setDistance(distance);
            vo.setDistanceDesc(formatDistance(distance));
            return vo;
        })
        .sorted(Comparator.comparing(HospitalVO::getDistance))
        .collect(Collectors.toList());
    
    // 4. 匹配蛇毒治疗关键词
    result.forEach(h -> {
        List<String> keywords = matchSnakeVenomKeywords(
            h.getHospitalName(),
            h.getDepartmentInfo()
        );
        h.setMatchedSnakeVenomKeywords(keywords);
    });
    
    return Result.success(result);
}

```javascript
// utils/api.js
import { useUserStore } from '@/store/user'

const BASE_URL = 'http://localhost:8888'

function request(options) {
  return new Promise((resolve, reject) => {
    const userStore = useUserStore()
    
    uni.request({
      url: BASE_URL + options.url,
      method: options.method,
      data: options.data,
      header: {
        'Content-Type': 'application/json',
        'Authorization': userStore.token ? `Bearer ${userStore.token}` : ''
      },
      success: (res) => {
        if (res.statusCode === 401) {
          // Token 过期，跳转登录
          uni.reLaunch({ url: '/pages/login/login' })
          return reject(new Error('未登录'))
        }
        
        if (res.data.code === 200) {
          resolve(res.data)
        } else {
          uni.showToast({
            title: res.data.message || '请求失败',
            icon: 'none'
          })
          reject(new Error(res.data.message))
        }
      },
      fail: (err) => {
        uni.showToast({
          title: '网络错误',
          icon: 'none'
        })
        reject(err)
      }
    })
  })
}

// user-service/controller/FileUploadController.java
@PostMapping("/upload")
public Result<Map<String, String>> uploadFile(
    @RequestParam("file") MultipartFile file) {
    
    if (file.isEmpty()) {
        return Result.error("文件为空");
    }
    
    // 校验文件大小 (不超过 20MB)
    if (file.getSize() > 20 * 1024 * 1024) {
        return Result.error("文件大小不能超过 20MB");
    }
    
    // 校验文件类型
    String contentType = file.getContentType();
    if (!contentType.startsWith("image/")) {
        return Result.error("只能上传图片文件");
    }
    
    try {
        // 生成文件名
        String fileName = UUID.randomUUID() + "_" + file.getOriginalFilename();
        String filePath = "uploads/" + fileName;
        
        // 保存到本地
        file.transferTo(new File(filePath));
        
        // 返回访问 URL
        Map<String, String> result = new HashMap<>();
        result.put("url", "/api/file/" + fileName);
        result.put("filename", fileName);
        
        return Result.success(result);
    } catch (IOException e) {
        log.error("文件上传失败", e);
        return Result.error("文件上传失败：" + e.getMessage());
    }
}

  url="${service#*:}"
  
  response=$(curl -s -o /dev/null -w "%{http_code}" "$url")
  
  if [ "$response" = "200" ]; then
    echo "✅ $name - 正常运行"
  else
    echo "❌ $name - 异常 (HTTP $response)"
  fi
done

echo "================================"
echo "检查完成"
```
